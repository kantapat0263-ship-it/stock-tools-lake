🚀 เริ่มต้นใช้งาน

3 สิ่งที่ควรรู้ก่อนเริ่ม

  • ใช้บนคอมพิวเตอร์ตั้งโต๊ะ เปิดเว็บ stock-tools-lake.vercel.app
  • เข้าระบบด้วย อีเมล + รหัสผ่าน ของตัวเอง (แต่ละคนมีบัญชีของตัวเอง)
  • 2 บริษัทใช้ระบบเดียวกัน — PAC และ LOTUS เห็น / ยืม / แก้ข้อมูลช่วยกันได้ทั้ง 2 บริษัท (ตั้งใจให้ช่วยกันทำงาน)

การเข้าใช้ครั้งแรก

  1. กด สมัครบัญชีใหม่ ที่หน้าเข้าระบบ → ใส่อีเมล + ตั้งรหัสผ่านของตัวเอง
  2. แจ้ง ผู้ดูแล (admin) ให้เปิดสิทธิ์ + กำหนดบริษัทให้คุณ
  3. เมื่อ admin ตั้งให้แล้ว ล็อกอินอีกครั้ง → จะเห็นข้อมูลทั้งหมด
⚠️ ถ้าล็อกอินแล้วเห็นว่าง ไม่มีข้อมูล แปลว่า admin ยังไม่ได้กำหนดบริษัทให้ — แจ้ง admin ได้เลย

🖥️ รู้จักหน้าจอ

  • เมนูด้านซ้าย — สลับไปแต่ละหน้า (Dashboard, ทะเบียน, เบิก/คืน, ส่งซ่อม ฯลฯ)
  • ตัวกรองบริษัท (มุมขวาบน) — ทั้งหมด PAC LOTUS เลือกเพื่อดูเฉพาะเครื่องของบริษัทที่ต้องการ
  • สีของระบบ — เปลี่ยนตามบริษัทของคุณ: PAC = น้ำเงินเข้ม, LOTUS = เขียวขี้ม้า
  • ตารางรายการยาว — เลื่อนดูในกล่อง หัวคอลัมน์ค้างบนสุดเสมอ (รู้ว่าข้อมูลอยู่ช่องไหน) + มีช่องค้นหาทุกหน้า
📱 ใช้บนมือถือได้: เปิดเว็บเดียวกันบนมือถือ → เมนูจะย่อเป็นปุ่ม ☰ (มุมซ้ายบน) แตะเพื่อเปิดเมนู (เหมาะกับการ "สแกน" ที่ออฟฟิศ)

🔖 รหัสเครื่อง = หมวด + NO

หัวใจของระบบ — เครื่องทุกตัวระบุด้วย "หมวด + เลข NO"

  • NO = เลขที่พ่น/เขียนไว้บนตัวเครื่องจริง
  • รูปแบบ NO ใหม่ = [บริษัท] - [โค้ดหมวด] - [เลขรัน] เช่น P-GR4-041 (P=PAC, GR4=หินเจียร 4 นิ้ว, 041=ลำดับ) หรือ L-MD-001 (L=LOTUS, MD=สว่านเจาะเหล็ก)
  • NO ห้ามซ้ำกัน — ถ้าใส่ NO ที่มีอยู่แล้ว ระบบจะเตือน ❌ NO นี้มีอยู่แล้ว และบันทึกไม่ได้ (กันสร้างซ้ำ)
  • เวลาค้นหา/เบิก/ส่งซ่อม ให้ดูที่ NO เป็นหลัก (S/N ที่ป้ายมักหลุดหาย เชื่อถือไม่ได้)
💡 เครื่องเก่าบางตัวยังเป็นเลขแบบเดิม (NO.01, NO.02 — ซ้ำข้ามหมวดได้) ค่อยๆ เปลี่ยนเป็นรูปแบบใหม่ตอนพ่น/ติดป้ายใหม่ · โค้ดหมวด (GR4/HD2/MD…) 1 หมวด 1 โค้ด ห้ามซ้ำกัน

📊 Dashboard (ภาพรวม)

หน้าแรกหลังเข้าระบบ — เห็นสรุปทุกอย่างในตาเดียว

  • การ์ดสรุป: เครื่องทั้งหมด / ว่าง / ถูกยืม / ส่งซ่อม
  • กราฟวงกลมแสดงสัดส่วนตามสถานะ
  • ตารางใบส่งซ่อมที่ยังค้างอยู่ + รายการเครื่องที่ถูกยืม
  • กิจกรรมล่าสุด (ใครทำอะไรไป)
🔔 แจ้งเตือนยืมข้ามบริษัท: ถ้าเครื่องของบริษัทคุณถูกอีกบริษัทยืมไป จะมีแถบเหลืองขึ้นเตือนตรงนี้ พร้อมบอกว่าเครื่องไหน ใครยืม อยู่ไซต์ไหน
📍 เตือนของค้างที่ไซต์: เครื่องที่เบิกไปไซต์แล้ว ค้างเกิน 30 วันยังไม่คืน จะถูกชูขึ้นเตือนตรงนี้อัตโนมัติ — กันลืม / กันหาย (ระบบจับเอง)
🔀 การโยกเครื่องล่าสุด: ถ้ามีการย้ายเครื่องไปไซต์ใน 7 วันล่าสุด (รวมที่ผู้ดูแลหน้างานสแกนย้ายเอง) จะมีแถบขึ้นบอก เครื่อง / ย้ายไปไหน / ใครย้าย / เมื่อไหร่ — ออฟฟิศเห็นทันที ไม่ต้องตามถาม

🧰 ทะเบียนเครื่องมือ

รายชื่อเครื่องมือทั้งหมด — ดู / ค้นหา / เพิ่ม / แก้ไข

ค้นหา & ดูรายละเอียด

  1. พิมพ์ในช่องค้นหา (พิมพ์ชื่อ / หมวด / NO ก็ได้)
  2. กรองด้วย หมวด · สถานะ · ไซต์งาน และเรียงลำดับได้ (ค่าเริ่มต้นเรียง เพิ่มล่าสุดก่อนเสมอ)
  3. คลิกที่แถวเครื่อง → เห็นรายละเอียดทั้งหมด + รูปสินค้า (ถ้ามี) · วันที่ซื้อ / ราคา / ประกัน · ประวัติการแก้ไข

เพิ่มเครื่องใหม่

  1. กดปุ่ม + เพิ่มเครื่อง
  2. เลือก บริษัทเจ้าของ · หมวด และ อยู่ที่ — เลือกจากรายการ (dropdown) ไม่พิมพ์เอง ถ้าไม่มีในรายการกด ➕ เพิ่มใหม่ได้
  3. กรอก NO (ห้ามซ้ำ — ซ้ำระบบเตือน) · ชื่อ · ยี่ห้อ · รุ่น · S/N
  4. กรอกเพิ่มได้: วันที่ซื้อ · ราคา (ใส่ comma ให้อัตโนมัติ เช่น 10,000) · ประกัน · อัปโหลด/ถ่ายรูปสินค้า (ไม่บังคับ)
  5. กดบันทึก — รหัสเครื่องสร้างอัตโนมัติจาก หมวด + NO
🖼️ รูปสินค้า: เครื่องส่วนใหญ่มีรูปขึ้นเองตามยี่ห้อ+รุ่น · ถ้ายังไม่มี/รูปผิด เปิดแก้เครื่อง → ช่อง "รูปสินค้า" อัปโหลด/ถ่ายรูปใส่เองได้ (อัป 1 ครั้งใช้กับทุกเครื่องรุ่นเดียวกัน)
🤝 แก้ได้ทั้ง 2 บริษัท: ทุกคนที่ล็อกอินสามารถ แก้ / เพิ่ม / ลบ เครื่องของ ทั้ง PAC และ LOTUS ได้ (ตั้งใจให้ช่วยกันทำงาน) — มีเฉพาะ "จัดการผู้ใช้" กับ "ประวัติแก้ไข" ที่สงวนไว้ให้ผู้ดูแล (admin)

🔄 เบิก / คืน

มี 3 แท็บ: เบิก · ย้ายไซต์ · คืน

💡 พอติ๊กเลือกเครื่อง ฟอร์มกรอกข้อมูล + ปุ่มบันทึกจะ โผล่ขึ้นด้านบน และลอยติดไว้ให้กดง่าย — ช่องแรกซ้ายสุดคือ "บริษัทที่ออกใบ (หัวกระดาษ)"

📤 เบิก (เอาออกไซต์)

  1. เลือกแท็บ เบิก → ติ๊กเครื่องที่ "ว่าง" ที่จะเอาออก (ติ๊กได้หลายตัว) → ฟอร์มจะโผล่ด้านบน
  2. เลือก "บริษัทที่ออกใบ (หัวกระดาษ)" = PAC หรือ LOTUS — หัวใบ/โลโก้ + เลขที่ใบ จะวิ่งตามบริษัทนี้
    เช่น คนโลตัสทำใบให้ PAC ก็เลือกหัว PAC ได้ เลขจะเข้าชุดของ PAC
  3. ในตารางแถวที่ติ๊ก กรอก เลขที่ PR และ หมายเหตุ ของเครื่องนั้นได้ (ไม่กรอกก็ได้ · เปลี่ยนหน้าแล้วค่าไม่หาย)
  4. กรอก วันที่ · ผู้เบิก · โครงการ/ไซต์ — ไซต์เลือกจากรายการ (ถ้าไม่มีกด ➕ เพิ่ม)
  5. กด 👁️ ดูใบส่งไซต์ ดูตัวอย่างใบเต็มก่อน → ถ้าถูก กด 🖨️ พิมพ์ + บันทึกเบิก → ออกใบส่งเครื่องมือ (2 สำเนา ตัดครึ่งเก็บฝั่งละใบ) + สถานะเป็น "ถูกยืม"
🧾 ใบส่งเครื่องมือ มีคอลัมน์ ลำดับ · รายการ · ยี่ห้อ · รุ่น · S/N · รหัส · เลขที่ PR · หมายเหตุ — PR/หมายเหตุ มาจากที่กรอกตอนเลือกเครื่อง
🔢 เลขที่ใบ รันต่อเนื่องอัตโนมัติ แยกแต่ละบริษัท (PAC ชุดหนึ่ง · LOTUS อีกชุดหนึ่ง)
🤝 เบิกปน 2 บริษัทในใบเดียว: สลับมุมบนเป็น "ทั้งหมด" ก่อน แล้วค่อยติ๊กเครื่องของทั้ง 2 บริษัท → ในประวัติจะเก็บเป็น ใบเดียว เหมือนตอนพิมพ์ (ไม่แตกเป็น 2)

📝 ใบร่าง (ทยอยเพิ่มเครื่อง)

ใช้ตอนยังไม่ออกใบจริงทันที — วันนึงมีคนทยอยมาเบิกเรื่อยๆ เก็บรวมไว้ก่อน แล้วค่อยออกใบทีหลัง

  1. ในแท็บ เบิก เลือกบริษัท/ผู้เบิก/ไซต์ + ติ๊กเครื่อง → กด 📝 เก็บเข้าร่าง (เครื่องเป็น "ถูกยืม" ทันที แต่ ยังไม่ออกเลข/ยังไม่พิมพ์)
  2. มีคนมาเบิกเพิ่ม ก็ติ๊กแล้วกด "เก็บเข้าร่าง" ซ้ำได้เรื่อยๆ
  3. ไปที่ ประวัติ → แท็บ 📝 ใบร่าง (มีตัวเลขบอกจำนวนใบร่าง) จะเห็นใบร่างทั้งหมด แต่ละใบมีปุ่ม:
ปุ่มทำอะไร
➕ เพิ่มเครื่องเด้งกลับไปแท็บเบิก โดยล็อกหัวใบให้แล้ว → ติ๊กเครื่องเพิ่ม → กด "เพิ่มเข้าใบร่างนี้"
✏️ แก้ไขแก้ เลขที่ PR / หมายเหตุ รายเครื่อง หรือเอาเครื่องออกจากใบร่าง
🖨️ ออกใบออกเลขที่ใบ + พิมพ์จริง → ใบร่างกลายเป็นใบสมบูรณ์ (ย้ายไปแท็บ "ใบเบิก/ส่งไซต์")
⚠️ เครื่องที่อยู่ในใบร่าง ถือว่า "ถูกยืม" ไปแล้ว (เห็นใน "ของตามไซต์") — แค่ยังไม่ได้ออกเลข/พิมพ์ใบ

🔀 ย้ายไซต์

  • ใช้เมื่อเครื่องที่ถูกยืมอยู่แล้ว ต้องย้ายไปไซต์ใหม่ โดยไม่ต้องคืนก่อน
  • ติ๊กเครื่อง → กรอกไซต์ใหม่ → กดบันทึก

📥 คืน (กลับออฟฟิต)

  • ติ๊กเครื่องที่ถูกยืมอยู่ → กรอกวันที่คืน → กดบันทึก → สถานะกลับเป็น "ว่าง"
ℹ️ ยืมเครื่องของอีกบริษัทได้ตามปกติ ระบบจะแจ้งเตือนเจ้าของให้รู้เองที่ Dashboard

📍 ของตามไซต์

เลือกไซต์งาน → เห็นว่ามีเครื่องอะไรอยู่ที่นั่นตอนนี้

  • ด้านบนเป็น ชิปไซต์ ที่มีของอยู่ + จำนวน (เรียงมากสุด) — กดเลือกไซต์
  • เห็นตารางเครื่องที่ไซต์นั้น: ชื่อ/รุ่น · NO · หมวด · ผู้ถือ · ออกเมื่อ · ค้างกี่วัน (เกิน 7 วัน = ส้ม, เกิน 30 = แดง เพื่อจับของค้างนาน)
  • คลิกแถวเครื่อง → ดูรายละเอียดได้
💡 ในหน้า ทะเบียนเครื่องมือ ก็มีตัวกรอง "ไซต์" ให้เลือกดูเฉพาะเครื่องของไซต์นั้นในตารางเดิมได้เช่นกัน

📤 ส่งซ่อม

สร้างใบส่งซ่อม → พิมพ์ส่งไปกับเครื่อง (เลือกหัวกระดาษ + มีใบร่าง เหมือนใบเบิก)

  1. ติ๊กเครื่องที่จะส่งซ่อม → ฟอร์มโผล่ด้านบน
  2. เลือก บริษัทที่ออกใบ (หัวกระดาษ) = PAC หรือ LOTUS · วันที่ · ร้าน/ช่างซ่อม (ขึ้น "ช่างสุเทพ" ให้อัตโนมัติ แก้ได้) · ผู้ส่ง
  3. รายเครื่องกรอกได้: โครงการ (เสียมาจากไซต์ไหน) + อาการ/สาเหตุ
  4. กด 👁️ ดูใบส่งซ่อม ดูตัวอย่างก่อน → ถ้าถูก กด 🖨️ พิมพ์ + บันทึกส่งซ่อม → ออก เลขที่ใบอัตโนมัติ (แยกตามบริษัท) + พิมพ์
✅ หัวใบแบบใหม่: เลขที่ตัวใหญ่มุมซ้ายบน + ชื่อบริษัทใต้หัวเรื่อง · คอลัมน์ โครงการ + อาการ ครบ · ให้ช่างเซ็นรับ — เครื่องเปลี่ยนเป็น "ส่งซ่อม"
📝 ใบร่างส่งซ่อม: กด 📝 เก็บเข้าร่าง แทน = ทยอยเก็บเครื่องไว้ก่อน (ยังไม่ออกเลข/ไม่พิมพ์) → ไปเช็ค + ออกใบทีหลังที่ ประวัติ → 📝 ใบร่าง

📥 รับคืนจากซ่อม

เครื่องซ่อมเสร็จกลับมา → กดรับเข้าสต็อกง่ายๆ (ไม่มีใบ ไม่ต้องพิมพ์)

  1. ดู "ใบส่งซ่อมที่ยังไม่ปิด" + รายการเครื่องที่อยู่ร้านซ่อม
  2. ติ๊กเครื่องที่ซ่อมเสร็จกลับมาแล้ว
  3. กรอกวันที่รับคืน + ผู้กดรับเข้า
  4. กด ✅ รับเข้า STOCK → สถานะกลับเป็น "ว่าง" พร้อมเบิกทันที
🚫 ออกใบผิด? ยกเลิกได้แบบปลอดภัย: กด "ยกเลิกใบ" → ต้อง พิมพ์เลขที่ใบยืนยัน (กันกดพลาด) → ใบจะ ย้ายเข้าถังขยะ — กู้คืนได้ ไม่ลบถาวร + ระบบคืนสถานะเครื่องให้อัตโนมัติ

📷 สแกน (มือถือ)

ส่อง QR ที่ติดเครื่อง → ปุ่มที่ขึ้นจะต่างกันตามสถานะเครื่อง + สิทธิ์ของคนสแกน

📲 สแกน QR (เด้งหน้าเต็มจอ)

  • ใช้กล้องมือถือส่อง QR บนตัวเครื่อง → เด้ง หน้าเต็มจอแยกจากเว็บ เห็นข้อมูล + ประวัติการเดินทาง + ค้างที่ไซต์กี่วัน
  • 🔎 สแกนไม่ติด / QR เสีย? ในหน้าสแกนมีช่อง พิมพ์ NO. หรือ S/N → เลือกเครื่องจากรายการ → ทำงานต่อได้เหมือนสแกนติด
  • ปุ่ม 📷 สแกนเครื่องอื่น เปิดกล้องส่องตัวถัดไปในหน้าเดิม · ครั้งแรกให้ ล็อกอินก่อน

ปุ่มที่ขึ้น — ตามสถานะ + สิทธิ์

สถานะ · คุณคือทำได้
ถูกยืม (อยู่ไซต์) · ออฟฟิศ📥 รับคืนเข้าออฟฟิต → ว่าง
ส่งซ่อม · ออฟฟิศ📥 รับคืนจากซ่อม → ว่าง + ปิดใบซ่อม
ว่าง · ออฟฟิศ📝 เก็บเข้าร่างส่งไซต์ (ดูด้านล่าง)
ถูกยืม · ผู้ดูแลหน้างาน🔀 ย้ายไปไซต์ (อย่างเดียว)
⚡ ยิงรัวได้ (เสร็จเด้งกล้องตัวถัดไป) · ↩️ ยกเลิก เผื่อกดผิด · ใส่โน้ตสภาพก่อนรับได้ · QR เล็ก มี แถบซูม ช่วย

📝 ออฟฟิศ — เก็บเครื่องเข้าร่างส่งไซต์ (สแกนในสโตร์)

  • อยู่ในสโตร์ หยิบเครื่อง "ว่าง" → สแกน → เลือก บริษัทที่ออกใบ + ไซต์ปลายทาง + PR (ไม่บังคับ) → กด เก็บเข้าร่างส่งไซต์
  • สแกนหลายตัวได้ — คนละไซต์แยกเป็นใบร่างคนละใบอัตโนมัติ (3 ไซต์ = 3 ใบร่าง) · ผู้เบิกเว้นไว้ ใส่ทีหลัง
  • ขึ้นออฟฟิศ (คอม) → ประวัติ → 📝 ใบร่าง → ใส่ผู้เบิก + เช็คให้เรียบร้อย → ออกใบ + พิมพ์ทีเดียว

🔀 ผู้ดูแลหน้างาน — สแกนย้ายไซต์

  • คนที่ตั้งบทบาท "ผู้ดูแลหน้างาน" สแกนได้อย่างเดียว: ย้ายเครื่องที่ถูกยืมอยู่ไปไซต์ใหม่ (เลือกไซต์ → กดย้าย)
  • ไม่มีปุ่มรับเข้า/แจ้งชำรุด · เปิดแอปออฟฟิศเต็มไม่ได้ (เห็นแต่หน้าสแกน)
  • พอย้าย → ออฟฟิศเห็นที่ Dashboard → 🔀 การโยกเครื่องล่าสุด ทันที

📜 ประวัติ

ไล่ดูการเคลื่อนไหวทั้งหมด — มี 6 แท็บ

แท็บใช้ทำอะไร
📋 รายการล่าสุดไล่ดูทุกความเคลื่อนไหว · กรองตามประเภท/ช่วงวันที่/คำค้น · จัดกลุ่มตามวัน · กด "โหลดเพิ่ม" ดูย้อนหลัง
📋 ใบเบิก / ส่งไซต์ใบส่งเครื่องมือที่ยังไม่คืน · มี ช่องค้นหา ด้านบน · แต่ละใบมีปุ่ม ✏️ แก้ไข (เพิ่ม/ลบเครื่อง, แก้ PR/หมายเหตุ, แก้หัวใบ) และ 🖨️ พิมพ์ ใบย้อนหลังได้
📝 ใบร่าง2 ส่วน: ใบร่างเบิก + ใบร่างส่งซ่อม · ทยอยเพิ่มเครื่อง ยังไม่ออกเลข/พิมพ์ · มีตัวเลขบอกจำนวน · แต่ละใบ ➕ เพิ่มเครื่อง / ✏️ แก้ไข / 🖨️ ออกใบ / 🗑️ ยกเลิก (คืนเครื่องเป็น "ว่าง")
📑 ใบส่งซ่อมใบส่งซ่อมทุกใบ ค้นด้วยเลขที่/ช่าง/ผู้ส่ง · คลิกที่ใบ → เห็นรายการเครื่อง + รับคืนแล้ว/ยัง · แต่ละใบมีปุ่ม ✏️ แก้ไข + 🖨️ พิมพ์ เช่นกัน · เรียงใบล่าสุดขึ้นก่อน (ตามวันที่ทำ)
🔧 ค้างซ่อม / เวลาซ่อมเครื่องที่ยังไม่กลับจากร้านซ่อม + ค้างมากี่วัน (สีแดง = นานเกินไป) และเครื่องที่ซ่อมเสร็จใช้เวลากี่วัน
🔍 รายเครื่องพิมพ์เลือกเครื่อง 1 ตัว → เห็นไทม์ไลน์ครบทั้งชีวิตของเครื่องนั้น
📝 ทุกรายการบอก เวลา และ "บันทึกโดย" (ใครเป็นคนคีย์เข้าระบบ)

📈 รายงาน

  • เห็นสรุปจำนวนเครื่องตามสถานะ + มูลค่ารวม
  • กด ⬇️ ดาวน์โหลด เพื่อโหลด ทะเบียนเครื่องมือ หรือ ประวัติ เป็นไฟล์ .csv (เปิดด้วย Excel ได้)
  • ไฟล์จะดาวน์โหลดตาม "ตัวกรองบริษัท" ที่เลือกอยู่มุมบนขวา

🏷️ ป้าย QR / TAG

สร้างป้ายติดเครื่อง (QR + ข้อมูล) — ดาวน์โหลด / พิมพ์ / เอาเข้าเครื่องพิมพ์ฉลาก NIIMBOT

  1. เข้าเมนู 🏷️ ป้าย QR → ติ๊กเลือกเครื่อง (หรือ "เลือกทั้งหมดที่เห็น")
  2. กด ⬇️ ป้าย ท้ายแถว → ได้ ป้ายเต็ม 5×3 ซม. เป็นรูป (QR + NO ตัวใหญ่ + ยี่ห้อ/รุ่น + S/N + ปีซื้อ + ตราบริษัท) เอาเข้าแอป NIIMBOT พิมพ์ติดเครื่องได้เลย
  3. หรือกด 🖨️ พิมพ์ / PDF ป้ายเต็ม (หลายดวง/แผ่น A4) · หรือ ⬇️ ป้าย PNG (ที่เลือก) โหลดทีละไฟล์

เช็คว่าติดป้ายเครื่องไหนแล้ว

  • มีช่อง "ติด TAG" — ติ๊กเมื่อแกะติดป้ายเครื่องนั้นแล้ว → ระบบนับให้ (ติดแล้ว X / ยังไม่ติด Y) + กรอง "เฉพาะยังไม่ติด" ได้
  • พิมพ์ทั้งล็อต (หลายร้อยตัว)? กด ⬇️ CSV ส่งออกรายการ + ลิงก์ QR → ทำ template พิมพ์ batch ในแอป NIIMBOT
🔢 บนป้ายมี NO ตัวใหญ่ — สแกนไม่ติดก็อ่านเลขจดได้ · QR เข้ารหัสแบบทนขูดขีด (ระดับ H)
💡 ติดที่ ตัวเครื่อง (ไม่ใช่กล่อง) แนะนำสติกเกอร์ทนๆ · QR ไม่ต้องทำใหม่ถ้าเปลี่ยนข้อมูล/เลข NO (ดูคำถามท้ายคู่มือ)

🗂️ จัดการไซต์งาน + หมวด

ดูแลรายชื่อ "ไซต์งาน" และ "หมวด (ประเภท)" ให้สะอาด ไม่ซ้ำ ไม่เพี้ยน

เดิมช่องไซต์/หมวดพิมพ์เองได้ → คนพิมพ์ต่างกันเลยแตกเป็นหลายชื่อ (เช่น "ฮอน ชิน" / "ฮอนชิน") · ตอนนี้ เลือกจากรายการอย่างเดียว + มี 2 เมนูไว้จัดระเบียบ:

🗂️ จัดการไซต์งาน

  • ➕ เพิ่ม ไซต์ใหม่
  • ✏️ แก้ชื่อ → ย้ายของ + ประวัติไปชื่อใหม่ให้
  • 🔀 รวม ไซต์ซ้ำ (เช่น ฮอนชิน → ฮอน ชิน)
  • 🗑️ ลบ ไซต์ที่ไม่ใช้แล้ว

📂 จัดการหมวด

  • หมวด = ประเภทกว้างๆ (สว่าน / หินเจียร / เลเซอร์…)
  • เพิ่ม / แก้ชื่อ / รวม / ลบ ได้เหมือนไซต์
  • ลบหมวดที่ยังมีเครื่องไม่ได้ — ต้องย้าย/รวมก่อน
💡 พอจัดเป็นรายการกลางแล้ว ทุกที่ (เบิก / ย้าย / แก้ทะเบียน) เลือกจากชุดเดียวกัน — ไม่งอกชื่อใหม่มั่วอีก

🗑️ ถังขยะ

เก็บทั้ง "เครื่องที่ถูกลบ" และ "ใบส่งซ่อมที่ยกเลิก" — กู้คืนได้หมด

  • 🗑️ ใบส่งซ่อมที่ยกเลิก — ใบที่กดยกเลิกจะมาอยู่ตรงนี้ กด ♻️ กู้คืนใบ เพื่อเอาใบ + ประวัติกลับเข้าระบบ
  • เครื่องที่ถูกลบ — ไม่หายจริง กด ♻️ กู้คืน เพื่อเอากลับมาได้

🔑 เมนูสำหรับผู้ดูแล (admin)

2 เมนูนี้เห็นเฉพาะผู้ดูแลระบบ

📝 ประวัติแก้ไข

ดูได้ทั้งระบบว่า ใคร / เมื่อไหร่ / แก้อะไร / เครื่องไหน — มีช่องค้นหา

👥 จัดการผู้ใช้

กำหนด บทบาท (พนักงาน / ผู้ดูแลหน้างาน / ผู้ดูแล) + บริษัท ของแต่ละคน · ลบผู้ใช้ได้

🔒 สมัครใหม่ = "⏳ รออนุมัติ" ยังเข้าไม่ได้: คนที่เพิ่งสมัครจะ เห็น/แก้ข้อมูลอะไรไม่ได้เลย จนกว่า admin จะเข้ามาตั้ง บทบาท + บริษัท ให้ (เห็นแต่หน้า "รออนุมัติ")
👷 บทบาท "ผู้ดูแลหน้างาน": สำหรับหัวหน้าไซต์ — สแกนย้ายเครื่องไปไซต์ได้อย่างเดียว ไม่เห็นเมนูออฟฟิศ ไม่มีปุ่มรับเข้า/แก้ทะเบียน

🎨 สถานะเครื่อง

สถานะความหมาย
ว่างอยู่ออฟฟิต พร้อมเบิก
ถูกยืมเบิกออกไปอยู่ไซต์งาน
ส่งซ่อมอยู่ที่ร้าน/ช่างซ่อม
รอส่งซ่อมเสียแล้ว รอส่งไปซ่อม

❓ คำถามที่พบบ่อย

ล็อกอินแล้วไม่เห็นข้อมูลเลย?

admin ยังไม่ได้กำหนดบริษัทให้ — แจ้ง admin

แก้ทะเบียนเครื่องของอีกบริษัทได้ไหม?

ได้เลย — ตอนนี้ทุกคนที่ล็อกอินแก้ได้ทั้ง PAC และ LOTUS (ตั้งใจให้ช่วยกันทำงาน)

ยืมเครื่องของอีกบริษัทได้ไหม?

ได้เลย ไม่ต้องขออนุญาตก่อน — ระบบจะแจ้งเตือนเจ้าของให้รู้เองที่ Dashboard

เผลอลบเครื่อง / ออกใบส่งซ่อมผิด?

ไม่หายถาวร กู้คืนได้หมดที่ "ถังขยะ" — ทั้งเครื่องที่ลบ และใบที่ยกเลิก (การยกเลิกใบต้องพิมพ์เลขใบยืนยันก่อน กันกดพลาด)

เปลี่ยนข้อมูล/เลข NO แล้ว QR เดิมยังสแกนได้ไหม?

ได้ตลอด — QR ผูกกับตัวเครื่องในระบบ (ไม่ใช่เลข NO) ต่อให้เปลี่ยนชื่อ / เลข NO / ย้ายบริษัท หรือลบเข้าถังขยะ ก็ยังสแกนเจอเครื่องเดิม · ข้อควรระวัง: เวลาเครื่องมีการเปลี่ยน ให้ "แก้ไข" รายการเดิม อย่าลบแล้วสร้างใหม่ (จะได้รหัสใหม่ QR เก่าจะใช้ไม่ได้)

สแกนมือถือทำอะไรได้?

ใช้ "รับของกลับเข้าระบบ" (จากไซต์ / จากซ่อม) เร็วๆ — ส่วนการส่งออกไซต์/ส่งซ่อมทำที่คอม

ลืมรหัสผ่าน?

แจ้งผู้ดูแลระบบ
