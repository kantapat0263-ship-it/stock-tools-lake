สารบัญ เริ่มต้นใช้งาน รู้จักหน้าจอ รหัสเครื่อง 📊 Dashboard 🧰 ทะเบียนเครื่องมือ 🔄 เบิก / คืน 📤 ส่งซ่อม 📥 รับคืนจากซ่อม 📷 สแกน (มือถือ) 📜 ประวัติ 📈 รายงาน 🏷️ ป้าย QR 🗑️ ถังขยะ 🔑 เมนูผู้ดูแล สถานะเครื่อง คำถามที่พบบ่อย

🚀 เริ่มต้นใช้งาน

3 สิ่งที่ควรรู้ก่อนเริ่ม

  • ใช้บนคอมพิวเตอร์ตั้งโต๊ะ เปิดเว็บ stock-tools-lake.vercel.app
  • เข้าระบบด้วย อีเมล + รหัสผ่าน ของตัวเอง (แต่ละคนมีบัญชีของตัวเอง)
  • 2 บริษัทใช้ระบบเดียวกัน — PAC และ LOTUS เห็นเครื่องมือของกันและกันได้ และ ยืมข้ามบริษัทได้

การเข้าใช้ครั้งแรก

  1. กด สมัครบัญชีใหม่ ที่หน้าเข้าระบบ → ใส่อีเมล + ตั้งรหัสผ่านของตัวเอง
  2. แจ้ง ผู้ดูแล (admin) ให้เปิดสิทธิ์ + กำหนดบริษัทให้คุณ
  3. เมื่อ admin ตั้งให้แล้ว ล็อกอินอีกครั้ง → จะเห็นข้อมูลทั้งหมด
⚠️ ถ้าล็อกอินแล้วเห็นว่าง ไม่มีข้อมูล แปลว่า admin ยังไม่ได้กำหนดบริษัทให้ — แจ้ง admin ได้เลย

🖥️ รู้จักหน้าจอ

  • เมนูด้านซ้าย — สลับไปแต่ละหน้า (Dashboard, ทะเบียน, เบิก/คืน, ส่งซ่อม ฯลฯ)
  • ตัวกรองบริษัท (มุมขวาบน) — ทั้งหมด PAC LOTUS เลือกเพื่อดูเฉพาะเครื่องของบริษัทที่ต้องการ
  • สีของระบบ — เปลี่ยนตามบริษัทของคุณ: PAC = น้ำเงินเข้ม, LOTUS = เขียวขี้ม้า
📱 ใช้บนมือถือได้: เปิดเว็บเดียวกันบนมือถือ → เมนูจะย่อเป็นปุ่ม ☰ (มุมซ้ายบน) แตะเพื่อเปิดเมนู (เหมาะกับการ "สแกน" ที่ออฟฟิศ)

🔖 รหัสเครื่อง = หมวด + NO

หัวใจของระบบ — เครื่องทุกตัวระบุด้วย "หมวด + เลข NO"

  • NO = เลขที่พ่น/เขียนไว้บนตัวเครื่องจริง (เช่น NO.02)
  • เลข NO ซ้ำกันได้ถ้าคนละหมวด เช่น "ตู้เชื่อม NO.02" กับ "หินเจียร์ NO.02" เป็นคนละตัว
  • เวลาค้นหา/เบิก/ส่งซ่อม ให้ดูที่ หมวด + NO เป็นหลัก (S/N ที่ป้ายมักหลุดหาย เชื่อถือไม่ได้)

📊 Dashboard (ภาพรวม)

หน้าแรกหลังเข้าระบบ — เห็นสรุปทุกอย่างในตาเดียว

  • การ์ดสรุป: เครื่องทั้งหมด / ว่าง / ถูกยืม / ส่งซ่อม
  • กราฟวงกลมแสดงสัดส่วนตามสถานะ
  • ตารางใบส่งซ่อมที่ยังค้างอยู่ + รายการเครื่องที่ถูกยืม
  • กิจกรรมล่าสุด (ใครทำอะไรไป)
🔔 แจ้งเตือนยืมข้ามบริษัท: ถ้าเครื่องของบริษัทคุณถูกอีกบริษัทยืมไป จะมีแถบเหลืองขึ้นเตือนตรงนี้ พร้อมบอกว่าเครื่องไหน ใครยืม อยู่ไซต์ไหน
📍 เตือนของค้างที่ไซต์: เครื่องที่เบิกไปไซต์แล้ว ค้างเกิน 30 วันยังไม่คืน จะถูกชูขึ้นเตือนตรงนี้อัตโนมัติ — กันลืม / กันหาย (ระบบจับเอง)

🧰 ทะเบียนเครื่องมือ

รายชื่อเครื่องมือทั้งหมด — ดู / ค้นหา / เพิ่ม / แก้ไข

ค้นหา & ดูรายละเอียด

  1. พิมพ์ในช่องค้นหา (พิมพ์ชื่อ / หมวด / NO ก็ได้)
  2. คลิกที่แถวเครื่อง → เห็นรายละเอียดทั้งหมด รวม วันที่ซื้อ / ราคา / ประกัน และประวัติการแก้ไข

เพิ่มเครื่องใหม่

  1. กดปุ่ม + เพิ่มเครื่อง
  2. เลือก บริษัทเจ้าของ (PAC / LOTUS) + กรอก หมวด, NO, ชื่อ, ยี่ห้อ ฯลฯ
  3. กรอกเพิ่มได้: วันที่ซื้อ / ราคา / ประกัน (ไม่บังคับ)
  4. กดบันทึก — รหัสเครื่องจะถูกสร้างอัตโนมัติจาก หมวด + NO
🔒 สิทธิ์การแก้ไข: แก้/เพิ่ม/ลบ ได้เฉพาะเครื่องของ บริษัทตัวเอง เท่านั้น (ของอีกบริษัทดูได้อย่างเดียว) — ยกเว้นผู้ดูแล (admin) ที่แก้ได้ทุกบริษัท

🔄 เบิก / คืน

มี 3 แท็บ: เบิก · ย้ายไซต์ · คืน

💡 พอติ๊กเลือกเครื่อง ฟอร์มกรอกข้อมูล + ปุ่มบันทึกจะ โผล่ขึ้นด้านบน และลอยติดไว้ให้กดง่าย

📤 เบิก (เอาออกไซต์)

  1. เลือกแท็บ เบิก → ติ๊กเครื่องที่ "ว่าง" ที่จะเอาออกไป (ติ๊กได้หลายตัว)
  2. กรอก วันที่ · ผู้เบิก · โครงการ/ไซต์ (บังคับครบ)
  3. กด 📤 บันทึกเบิก → สถานะเครื่องเปลี่ยนเป็น "ถูกยืม"

🔀 ย้ายไซต์

  • ใช้เมื่อเครื่องที่ถูกยืมอยู่แล้ว ต้องย้ายไปไซต์ใหม่ โดยไม่ต้องคืนก่อน
  • ติ๊กเครื่อง → กรอกไซต์ใหม่ → กดบันทึก

📥 คืน (กลับออฟฟิต)

  • ติ๊กเครื่องที่ถูกยืมอยู่ → กรอกวันที่คืน → กดบันทึก → สถานะกลับเป็น "ว่าง"
ℹ️ ยืมเครื่องของอีกบริษัทได้ตามปกติ ระบบจะแจ้งเตือนเจ้าของให้รู้เองที่ Dashboard

📤 ส่งซ่อม

สร้างใบส่งซ่อม → พิมพ์ส่งไปกับเครื่อง

  1. ติ๊กเครื่องที่จะส่งซ่อม + กรอก อาการ/สาเหตุ ของแต่ละตัว
  2. กรอกวันที่ส่ง · ร้าน/ช่างซ่อม · ผู้ส่ง
  3. กด 👁️ ดูใบส่งซ่อม เพื่อ ดูตัวอย่างก่อน (ยังไม่บันทึก)
  4. ถ้าถูกต้อง กด 🖨️ พิมพ์ + บันทึกส่งซ่อม → ระบบออก เลขที่ใบอัตโนมัติ + เปิดหน้าพิมพ์
✅ พิมพ์ใบออกมาส่งไปพร้อมเครื่อง ให้ช่างเซ็นรับ — สถานะเครื่องเปลี่ยนเป็น "ส่งซ่อม"
💡 ถ้าแค่อยากดูหน้าตาใบ ยังไม่ส่งจริง — กดดูใบแล้วปิดได้เลย จะไม่บันทึกอะไร

📥 รับคืนจากซ่อม

บันทึกเครื่องที่ซ่อมเสร็จกลับมา → พร้อมเบิกต่อ

  1. ดูรายการ "ใบส่งซ่อมที่ยังไม่ปิด" ด้านบน
  2. ติ๊กเครื่องที่ซ่อมเสร็จกลับมาแล้ว (จากรายการเครื่องที่อยู่ร้านซ่อม)
  3. กรอกวันที่รับคืน + ผู้เซ็นรับคืน
  4. กด ✅ บันทึกรับคืน + พิมพ์ → สถานะกลับเป็น "ว่าง" พร้อมเบิก
🚫 ออกใบผิด? กดปุ่ม "ยกเลิกใบ" ที่รายการใบส่งซ่อมได้ — ระบบจะคืนสถานะเครื่องให้อัตโนมัติ

📷 สแกน (มือถือ)

ใช้บนมือถือที่ออฟฟิศ — รับของกลับเข้าระบบเร็วๆ ด้วยการสแกน QR

  1. เข้าเมนู 📷 สแกน → กด เปิดกล้อง → ส่อง QR บนตัวเครื่อง (หรือพิมพ์รหัส/NO ก็ได้)
  2. เครื่องจะเด้งขึ้นมาพร้อมปุ่มตามสถานะ:
เครื่องอยู่...ปุ่มที่ขึ้น
ไซต์งาน (ถูกยืม)📥 รับคืนเข้าออฟฟิต → กลายเป็น "ว่าง"
ร้านซ่อม📥 รับคืนจากซ่อม → ว่าง + ปิดใบส่งซ่อมให้
ว่างแล้ว (ที่ออฟฟิต)ไม่ต้องทำอะไร
⚡ กดเสร็จเด้งกลับไปสแกนตัวต่อไปได้เลย — ยิงรัวๆ หลายเครื่องได้ ไม่ต้องกรอกอะไร · มี ⚠️ แจ้งชำรุด เผื่อเจอเครื่องเสีย
หน้าสแกนใช้สำหรับ "รับของเข้า" เท่านั้น — การ ส่งไปไซต์ / ส่งซ่อม ยังทำที่คอม (เมนู เบิก/คืน และ ส่งซ่อม) เพราะต้องกรอกรายละเอียด + พิมพ์ใบ

📜 ประวัติ

ไล่ดูการเคลื่อนไหวทั้งหมด — มี 3 แท็บ

แท็บใช้ทำอะไร
📋 รายการล่าสุดไล่ดูทุกความเคลื่อนไหว · กรองตามประเภท/ช่วงวันที่/คำค้น · จัดกลุ่มตามวัน · กด "โหลดเพิ่ม" ดูย้อนหลัง
🔧 ค้างซ่อม / เวลาซ่อมเครื่องที่ยังไม่กลับจากร้านซ่อม + ค้างมากี่วัน (สีแดง = นานเกินไป) และเครื่องที่ซ่อมเสร็จใช้เวลากี่วัน
🔍 รายเครื่องพิมพ์เลือกเครื่อง 1 ตัว → เห็นไทม์ไลน์ครบทั้งชีวิตของเครื่องนั้น
📝 ทุกรายการบอก เวลา และ "บันทึกโดย" (ใครเป็นคนคีย์เข้าระบบ)

📈 รายงาน

  • เห็นสรุปจำนวนเครื่องตามสถานะ + มูลค่ารวม
  • กด ⬇️ ดาวน์โหลด เพื่อโหลด ทะเบียนเครื่องมือ หรือ ประวัติ เป็นไฟล์ .csv (เปิดด้วย Excel ได้)
  • ไฟล์จะดาวน์โหลดตาม "ตัวกรองบริษัท" ที่เลือกอยู่มุมบนขวา

🏷️ ป้าย QR

สร้าง QR ของเครื่องแต่ละตัว (ไม่ซ้ำกัน) เอาไปทำป้ายติดเครื่อง

  1. เข้าเมนู 🏷️ ป้าย QR → ติ๊กเลือกเครื่อง (หรือกด "เลือกทั้งหมดที่เห็น")
  2. กด 🖨️ พิมพ์ / บันทึก PDF → ได้แผ่นป้าย QR (4 ดวง/หน้า) เอาไปปริ้นติดเครื่อง
  3. หรือกด ⬇️ QR ท้ายแถว เพื่อดาวน์โหลด QR ทีละตัวเป็นรูป (.png)
💡 ป้าย QR ใช้คู่กับเมนู 📷 สแกน — ติดที่ ตัวเครื่อง (ไม่ใช่กล่อง) แนะนำสติกเกอร์กันน้ำ/ป้ายทนๆ จะได้ไม่หลุดง่าย

🗑️ ถังขยะ

  • เครื่องที่ถูกลบจะ ไม่หายจริง — มาอยู่ในถังขยะ
  • กดปุ่ม ♻️ กู้คืน เพื่อเอากลับมาได้ (กู้คืนได้เฉพาะบริษัทตัวเอง)

🔑 เมนูสำหรับผู้ดูแล (admin)

2 เมนูนี้เห็นเฉพาะผู้ดูแลระบบ

📝 ประวัติแก้ไข

ดูได้ทั้งระบบว่า ใคร / เมื่อไหร่ / แก้อะไร / เครื่องไหน — มีช่องค้นหา

👥 จัดการผู้ใช้

กำหนดบทบาท (พนักงาน/ผู้ดูแล) + บริษัท ของแต่ละคน และลบผู้ใช้ได้

เมื่อมีพนักงานใหม่สมัครบัญชี ให้ admin เข้ามาที่เมนูนี้เพื่อ กำหนดบริษัท ให้เขาก่อน เขาถึงจะเห็นข้อมูล

🎨 สถานะเครื่อง

สถานะความหมาย
ว่างอยู่ออฟฟิต พร้อมเบิก
ถูกยืมเบิกออกไปอยู่ไซต์งาน
ส่งซ่อมอยู่ที่ร้าน/ช่างซ่อม
รอส่งซ่อมเสียแล้ว รอส่งไปซ่อม
ชำรุด / สูญหายใช้ไม่ได้ / หาไม่เจอ

❓ คำถามที่พบบ่อย

ล็อกอินแล้วไม่เห็นข้อมูลเลย?

admin ยังไม่ได้กำหนดบริษัทให้ — แจ้ง admin

แก้ทะเบียนเครื่องของอีกบริษัทไม่ได้?

ถูกต้องแล้ว — แก้ได้เฉพาะบริษัทตัวเอง (ของอีกบริษัทดูได้อย่างเดียว)

ยืมเครื่องของอีกบริษัทได้ไหม?

ได้เลย ไม่ต้องขออนุญาตก่อน — ระบบจะแจ้งเตือนเจ้าของให้รู้เองที่ Dashboard

เผลอลบเครื่อง / ออกใบส่งซ่อมผิด?

ลบเครื่อง → กู้คืนได้ที่ "ถังขยะ" · ออกใบผิด → กด "ยกเลิกใบ" ที่หน้ารับคืนจากซ่อม

สแกนมือถือทำอะไรได้?

ใช้ "รับของกลับเข้าระบบ" (จากไซต์ / จากซ่อม) เร็วๆ — ส่วนการส่งออกไซต์/ส่งซ่อมทำที่คอม

ลืมรหัสผ่าน?

แจ้งผู้ดูแลระบบ
